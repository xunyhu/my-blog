# JavaScript 异步编程

## 一、是什么

JS 是单线程：

👉 通过异步机制处理耗时任务

## 二、异步方式演进

回调函数 → Promise → async/await

## 三、Promise 原理

### 状态

- pending
- fulfilled
- rejected

### 示例

```js id="promise_example"
const p = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve(1);
  }, 1000);
});
```

## 四、async/await

```js
async function test() {
  const res = await fetch('/api');
  console.log(res);
}
```

👉 本质是 Promise 语法糖

## 五、总结

异步本质：`“延迟执行 + 回调通知机制”`
