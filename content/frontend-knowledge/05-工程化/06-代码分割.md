# 代码分割

## 一、是什么

将代码拆分成多个 bundle，实现按需加载。

## 二、实现方式

1. 动态 import

```js
import('./module.js');
```

2. 路由分割（React）

```js
const Home = React.lazy(() => import('./Home'));
```

3. SplitChunks

```js
optimization: {
  splitChunks: {
    chunks: 'all';
  }
}
```

## 三、优势

- 减少首屏加载
- 提升性能

## 四、总结

代码分割本质：

按需加载 + 延迟执行
