# HTTP协议

## 一、是什么

HTTP（超文本传输协议）是客户端和服务器之间的通信协议。

## 二、特点

- 无状态
- 明文传输
- 基于请求-响应

## 三、请求结构

GET /index.html HTTP/1.1

Host: example.com

## 四、响应结构

HTTP/1.1 200 OK

Content-Type: text/html

## 五、常见方法

- GET（获取）
- POST（提交）
- PUT（更新）
- DELETE（删除）

## 六、状态码

| 状态码 | 含义       |
| ------ | ---------- |
| 200    | 成功       |
| 301    | 重定向     |
| 404    | 未找到     |
| 500    | 服务器错误 |
