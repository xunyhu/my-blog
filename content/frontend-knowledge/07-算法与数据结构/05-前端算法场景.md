## 2. 防抖与节流

防抖（Debounce）

```js
function debounce(fn, delay) {
  let timer;
  return function () {
    clearTimeout(timer);
    timer = setTimeout(fn, delay);
  };
}
```

节流（Throttle）

```js
function throttle(fn, delay) {
  let last = 0;
  return function () {
    const now = Date.now();
    if (now - last > delay) {
      fn();
      last = now;
    }
  };
}
```

- DFS / BFS

- 树转数组

```js
function flatten(tree) {
  return tree.reduce((res, node) => {
    return res.concat(node, flatten(node.children || []));
  }, []);
}
```
